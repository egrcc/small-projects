xunmi_android
=============

"寻觅"是一个搜索网盘资源的android app，目前支持百度网盘，微盘，华为网盘。支持android4.0及以上。开发该应用的初衷只是为了能够在移动端上便捷快速的搜索到自己想要的资源。「寻觅」并不是一款搜索引擎，所有搜索结果均来源于第三方搜索引擎。其中百度网盘和华为网盘的搜索结果由<a href="http://www.gfsoso.com/">谷粉搜搜</a>提供，本程序已获得谷粉开发者的授权，也希望大家多多支持他们的网站。微盘的搜索结果由微盘官网的搜索引擎提供。下载链接：http://www.wandoujia.com/apps/com.egrcc.search

## 使用方法
下载整个项目，直接导入eclipse即可，不依赖于其他项目。

## 更新历史
### 1.0.2(2014-10-19)

- 获得谷粉授权，正式启用谷粉引擎。
- 进一步优化百度网盘的搜索结果，可以搜索到“yun.baidu.com”的资源。
- 增加长按复制链接的功能。
- 修复输入法频繁弹出的bug。
- 修复在部分机型上菜单键不显示的bug。

### 1.0.1(2014-10-18)

- 优化百度网盘的搜索结果，解决部分机型搜不到的bug。

### 1.0.0(2014-10-13)

- 完成「寻觅」主体程序的编写，准备发布。
- 支持百度网盘，微盘，华为网盘的搜索功能。
